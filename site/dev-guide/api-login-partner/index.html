    Document API Login¶

    1. API Đăng nhập¶

    POST: /api/partner/login

    Khởi tạo phiên làm việc mới cho người dùng B2B2C

    Chú ý

    Yêu cầu bảo mật: Mã hóa dữ liệu và kèm theo chữ ký điện tử

    Request Body¶

    Model

    • key (string, required),

      Key giải mã dữ liệu (đã được mã hóa). Cách thành lập tham khảo mục: Mã hóa dữ liệu truyền và xác thực chữ ký điện tử

    • data (string, required),

      Dữ liệu kèm theo chữ ký điện tử (đã được mã hóa). Cách giải mã tham khảo mục: Mã hóa dữ liệu truyền và xác thực chữ ký điện tử

      Signature data schema:

      <access_code>|<user_id>

      Original data schema: 

      <access_code>|<user_id>|<signature>

      • access_code (String, required)

        Access code do Gotadi cung cấp cho Đối tác.

      • user_id (String, required)

        Mã định danh người dùng trên hệ thống của Đối tác.

    {
    "data": "string",
    "key": "string"
}

    Response¶

    Code 200¶

    OK

    Model

    • key (String, required)

      Key giải mã dữ liệu (đã được mã hóa). Cách giải mã tham khảo mục: Mã hóa dữ liệu truyền và xác thực chữ ký điện tử

    • data (String, required)

      Dữ liệu kèm theo chữ ký điện tử (đã được mã hóa). Cách giải mã tham khảo mục: Mã hóa dữ liệu truyền và xác thực chữ ký điện tử

      Signature data schema:

      <access_code>|<error_code>

      Original data schema: 

      <jwt_token>|<error_code>|<signature>

      • access_code (String, required)

        Access code do Gotadi cung cấp cho Đối tác.

      • error_code (String, required)

        Mã lỗi

      • jwt_token (String, required)

        token dùng để gọi api của gotadi

    {
    "data": "string",
    "key": "string"
}

      Email khách hàng

      Thành phố

    • cityCode (string, required),

    • country (string, required),

      Quốc gia

      Mã quốc gia

    • firstName (string, required),

      Tên khách hàng

    • lastName (string, required),

      Họ khách hàng

    • phoneNumber (string, required),

      Số điện thoại khách hàng
